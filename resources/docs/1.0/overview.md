## Prerequisite
 
---

- Docker
- Docker-compose
- Web Browser (Google Chrome / Mozilla Firefox) 
- Open Ports
    - 9000: PHP
    - 8080: Nginx
    - 3307: MySQL
    - 6382: Redis
